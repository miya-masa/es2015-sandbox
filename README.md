# ES2015の基礎
## 準備

1. nodeをインストールする https://nodejs.org/en/
2. gitをインストールする https://git-for-windows.github.io/
3. 任意のディレクトリで以下のコマンドを実行

  ```txt
   git clone https://github.com/miya-masa/es2015-sandbox.git
   cd es2015-sandbox
   git checkout study
   npm install -g gulp
   npm install
   npm run start
  ```
4. ブラウザが立ち上がれば準備完了
